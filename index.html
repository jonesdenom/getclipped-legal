How to use getClipped

This is a one-time setup. The goal is to make sure getClipped and OBS Studio both save recordings/replays to the exact same folders.

  1. In getClipped, go to Settings → Recording & Replay Folders and set your recording folder.
  2. Match this with your OBS Studio recording path under Output → Recording → Recording Path.
  3. Repeat the same for Replay Buffer (usually the same folder).
  4. Ensure the Replay Buffer is active (blue highlight under your stream controls).
  5. Start clipping! Any replay or short recording you make will automatically be edited and returned to the app.

Optional: Set up hotkeys in OBS for starting/stopping recordings and saving replays to make the process one button.
