How to use getClipped

Set up OBS to drop replays/recordings into the same folder your desktop app watches, then clip and upload directly.

  1. In getClipped desktop, note your replay/recording folder in Settings.
  2. In OBS → Output → Recording, set the Recording Path to that same folder.
  3. Enable Replay Buffer in OBS (set duration to the clip length you want).
  4. While live, save a replay — it will appear inside your desktop app ready to process.

Streamlabs Desktop can do the same thing — just point Streamlabs recordings to the same folder.

  1. Open Streamlabs → Settings → Output → Recording and set the path to your getClipped folder.
  2. Make a test recording while streaming.
  3. Open the desktop app → Upload tab and you should see the clip ready.

Aitum Vertical already gives you vertical capture — this shows how to feed those captures into getClipped.

  1. Export your Aitum Vertical capture to the watched folder.
  2. Open getClipped desktop.
  3. Choose the new file to process and upload to TikTok, Shorts or Reels.

Paid users can add their own branding, transitions and caption colours directly inside the desktop app.

  1. In the desktop app, go to Settings → Branding / Stinger.
  2. Upload your own transition/stinger.
  3. Set custom caption colours and fonts.
  4. Process another clip and your branding will be baked in automatically.
